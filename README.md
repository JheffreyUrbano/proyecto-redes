# Proyecto Redes

## Cómo levantar el ambiente
1. Instalar [Vagrant](https://vagrantup.com) y [VirtualBox](https://virtualbox.org)
2. Clonar este repo: `git clone https://github.com/JheffreyUrbano/proyecto-redes.git`
3. Ir a la carpeta: `cd proyecto-rei-2026`
4. Levantar las VMs: `vagrant up`
   - Las boxes se descargan automáticamente
5. Conectarse: `vagrant ssh servidorUbuntu1`
